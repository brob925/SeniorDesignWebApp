Choose what node and sensor to view

Using the drop down menus on the top bar, it's simple to change which node and which sensor to view data from. You can even specify whether to see the data in an interactive graphic format or to just view the data in a table.

Select your own time frame

At the bottom of each graph, map, and table you can select a time frame of the data collected to view. This allows you to zoom in and focus on the data you care about or to see the big picture.
