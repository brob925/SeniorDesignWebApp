Get to know your WSN

Powerful Senses

Tons of measurements

Each sensor node is equiped with many different sensors for detecting various elements of the surrounding environment. All the sensors are very sensative so you can expect very precise measurements.

Power that Lasts

Don't worry about a dead node

You can expect your sensors to stay alive as their power source is recharged by the sun. Using the power of solar energy your node can last as long as you need it to.

Wireless Communication

Transmission through the power of HSPA

The sensor data is sent to the server safely and securingly using the same techonology as your every day smartphone. Not only do the nodes talk to server but they also talk to one another so that they can better adapt to the surrounding enviroment.

View your Data. From Anywhere.

All you need is an internet browser

Using the well developed web application you can easily view ALL of the data your sensor has collected. The application even allows you select what dates you would like to view your data so that you can easily see trends throughout the day, week, or even month.
